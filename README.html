# Demo Site [Parking App](http://demonic-cattle.surge.sh/) # Parking App A JQuery app that consumes a [Philadelphia Parking API](http://phlapi.com/airportparking.html) to display the user information about where to park on their upoming travels. This app also makes use of the HTML5 Geolocation to provide the use with directions from their current location. ## Frameworks & Libraries - JQuery - Simplify API requests and DOM manipulation - Animate.css - Add animations to DOM content ## Ideas for Improvement
